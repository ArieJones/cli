# Usage
<!-- usage -->
```sh-session
$ npm install -g nhost-cli
$ nhost-cli COMMAND
running command...
$ nhost-cli (-v|--version|version)
nhost-cli/0.0.0 darwin-x64 node-v12.16.1
$ nhost-cli --help [COMMAND]
USAGE
  $ nhost-cli COMMAND
...
```
