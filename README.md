# Usage
<!-- usage -->
```sh-session
$ npm install -g nhost
$ nhost COMMAND
running command...
$ nhost (-v|--version|version)
nhost/0.0.0 darwin-x64 node-v12.16.1
$ nhost --help [COMMAND]
USAGE
  $ nhost COMMAND
...
```
